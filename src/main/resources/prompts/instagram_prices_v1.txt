- 설명, 요약, 주석, 코드블록, 마크다운, 자연어 문장은 절대 출력하지 마세요.
- JSON 외의 어떤 텍스트도 출력하면 안 됩니다.

━━━━━━━━━━━━━━━━━━━━━━
🎯 출력 포맷 (절대 변경 금지)
━━━━━━━━━━━━━━━━━━━━━━
[
  {
    "type": "ENUM_STYLE_STRING",
    "price": 0,
    "info": "한글 설명",
    "condition": "조건 설명"
  }
]

━━━━━━━━━━━━━━━━━━━━━━
📌 필드 규칙 (엄격)
━━━━━━━━━━━━━━━━━━━━━━
1. type
- 영문 대문자 + SNAKE_CASE
- category + 콘텐츠 타입 결합
- 예시:
  - REPOST_IMAGE
  - REPOST_REELS
  - AD_IMAGE
  - ONLINE_EVENT_REELS
  - KAKAO_PUSH_IMAGE

2. price
- 숫자만 허용 (단위, 콤마 제거)
- 가격이 없거나 인식 불가 시 null

3. info
- 사람이 읽기 쉬운 한글 설명
- category + 타입을 자연어로 표현
- 예: "리포스트 이미지", "온라인몰 기획전 릴스"

━━━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━━━
- JSON 외 출력
- 필드 추가/삭제
- 값 보정, 계산, 추정

━━━━━━━━━━━━━━━━━━━━━━
